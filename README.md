- [ ] Anime search
- [ ] Manga search
- [ ] Anime schedule
- [ ] Anime and manga recommendations
- [ ] User profile and authentication
- [ ] User settings

## Installation
### Setup
```bash
git clone https://github.com/EverythingSuckz/HanabiBot
cd HanabiBot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
### Configuration
```bash
nano .env
```

### Running
```bash
python -m Hanabi
```
